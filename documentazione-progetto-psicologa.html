Panoramica

Il progetto è un'applicazione single-page (SPA) realizzata con Angular, pensata per presentare la figura professionale della psicologa, i servizi che offre e facilitare il contatto diretto con l'utente.

L'obiettivo è creare un sito empatico, accessibile e funzionale, con una navigazione semplice e un'estetica coerente con i valori di ascolto, empatia e benessere interiore.

Overview

The project is a single-page application (SPA) built with Angular, designed to present the professional profile of the psychologist, the services offered, and facilitate direct contact with the user.

The goal is to create an empathetic, accessible, and functional site with simple navigation and aesthetics consistent with the values of listening, empathy, and inner well-being.

Tecnologie Utilizzate

  • Angular 17+ (con componenti standalone)
  • TypeScript
  • Tailwind CSS
  • HTML5
  • Router di Angular

Design e UI

  • UI progettata con Tailwind CSS.
  • Layout flessibile e ottimizzato per diversi dispositivi.
  • Classi personalizzate per gestione sfondi e sezioni tematiche.
  • Palette colori neutra con accenti pastello per tono empatico.

Design and UI

  • UI designed with Tailwind CSS.
  • Flexible layout optimized for different devices.
  • Custom classes for background management and thematic sections.
  • Neutral color palette with pastel accents for an empathetic tone.

Funzionalità Implementate

  • Navigazione fluida e SPA tramite RouterLink.
  • Componenti standalone e modulari.
  • Form contatto con invio mail tramite EmailJS.
  • Responsive completo (mobile, tablet, desktop).
  • Sezioni su disturbi psicologici con approccio divulgativo.

Implemented Features

  • Smooth SPA navigation via RouterLink.
  • Standalone and modular components.
  • Contact form with email sending with EmailJS Service.
  • Full responsive (mobile, tablet, desktop).
  • Sections on psychological disorders with a divulgative approach.

Deploy

Il progetto è stato distribuito online utilizzando il servizio di hosting Netlify, garantendo una pubblicazione veloce e affidabile con aggiornamenti continui tramite integrazione con il repository Git.

Il sito è disponibile all'indirizzo: giuliaspanupsicologa.com

Il codice sorgente è disponibile su GitHub: https://github.com/VeronicaDiGiac/Giulia-Spanu-Psicologa
