Panoramica

Il progetto è un'applicazione meteo realizzata con Angular che permette di visualizzare le condizioni meteorologiche attuali e le previsioni per i giorni successivi per qualsiasi città nel mondo.

L'applicazione utilizza le API di Open-Meteo per ottenere dati precisi su temperatura, umidità, precipitazioni, vento, e l'adattamento dell'interfaccia in base all'ora del giorno (modalità giorno/notte).

Overview

The project is a weather application built with Angular that displays current weather conditions and forecasts for upcoming days for any city worldwide.

The application uses Open-Meteo APIs to get accurate data on temperature, humidity, precipitation, wind, and the interface adaptation based on the time of day (day/night mode).

Tecnologie Utilizzate

  • Angular 17+ (componenti standalone)
  • TypeScript
  • RxJS (per la gestione degli Observable)
  • API Open-Meteo (geocoding e dati meteo)
  • BehaviorSubject per lo stato globale
  • CSS moderno con variabili custom

Technologies Used

  • Angular 17+ (standalone components)
  • TypeScript
  • RxJS (for Observable management)
  • Open-Meteo APIs (geocoding and weather data)
  • BehaviorSubject for global state
  • Modern CSS with custom variables

Componenti Principali

Componente Descrizione
WeatherCardComponent Componente principale che gestisce la logica dell'applicazione e coordina gli altri componenti.
MeteoDetailsComponent Visualizza i dettagli meteo principali (temperatura, umidità, etc.) con animazioni di entrata.
NavbarComponent Barra di ricerca per selezionare una nuova città.
DayDetailsComponent Mostra le previsioni giornaliere con temperatura min/max e orari di alba/tramonto.
ApiService Servizio che gestisce tutte le chiamate API e lo stato globale della città selezionata.
RoundTempPipe Pipe personalizzata per arrotondare i valori di temperatura.

Main Components

Component Description
WeatherCardComponent Main component that manages application logic and coordinates other components.
MeteoDetailsComponent Displays main weather details (temperature, humidity, etc.) with entry animations.
NavbarComponent Search bar to select a new city.
DayDetailsComponent Shows daily forecasts with min/max temperature and sunrise/sunset times.
ApiService Service that manages all API calls and global state of selected city.
RoundTempPipe Custom pipe to round temperature values.

Design and UI

  • Clean and minimalist interface focused on essential data
  • Responsive layout that adapts to all devices
  • Smooth animations for component entry
  • Automatic day/night mode based on local time
  • Intuitive card design for daily forecasts
  • Clear icons for different weather parameters

Funzionalità Implementate

  • Ricerca città con geocoding per ottenere coordinate precise
  • Visualizzazione dati meteo in tempo reale
  • Previsioni giornaliere per i prossimi giorni
  • Gestione dello stato globale della città selezionata
  • Salvataggio locale dell'ultima città cercata
  • Adattamento automatico all'ora del giorno (tema chiaro/scuro)
  • Animazioni per migliorare l'esperienza utente
  • Gestione degli errori per chiamate API fallite
  • Formattazione personalizzata dei dati (es. arrotondamento temperature)

Implemented Features

  • City search with geocoding for precise coordinates
  • Real-time weather data display
  • Daily forecasts for upcoming days
  • Global state management of selected city
  • Local storage of last searched city
  • Automatic adaptation to time of day (light/dark theme)
  • Animations to enhance user experience
  • Error handling for failed API calls
  • Custom data formatting (e.g. temperature rounding)
