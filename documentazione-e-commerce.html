Panoramica

Il progetto è un'applicazione e-commerce sviluppata con Angular che offre una piattaforma completa per la vendita di abbigliamento con sezioni dedicate per uomini e donne.

L'applicazione include tutte le funzionalità principali di un e-commerce: carrello, checkout, gestione del profilo utente, newsletter e integrazione con database esterno per la gestione dei prodotti.

Il progetto è stato deployato su GitHub Pages e dimostra una solida implementazione dei concetti fondamentali di Angular, tra cui gestione dello stato, componenti riutilizzabili e routing avanzato.

Overview

The project is an e-commerce application built with Angular that provides a complete platform for clothing sales with dedicated sections for men and women.

The application includes all main e-commerce features: shopping cart, checkout, user profile management, newsletter and integration with external database for product management.

The project has been deployed on GitHub Pages and demonstrates a solid implementation of fundamental Angular concepts including state management, reusable components and advanced routing.

Tecnologie Utilizzate

  • Angular 17+ (componenti standalone)
  • TypeScript
